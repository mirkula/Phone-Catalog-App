# Starter

## Installation Server

``` bash
$ cd server                     
# install dependencies
$ npm install
```

## Installation Client

``` bash
$ cd client                     
# install dependencies
$ npm install
```

## Usage Server

### Development

``` bash
$ node index.js
```

## Usage Client

### Development

``` bash
# serve with hot reloading at localhost:3000
$ npm start
```

Go to [http://localhost:3000](http://localhost:3000)

### Production

``` bash
# build for production and launch the server
$ npm run build
```

### Test

``` bash
$ npm test
```
